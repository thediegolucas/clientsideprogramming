HTML

HTML is the standard markup language for web pages. It stands for Hyper-Text Markup Language and it is used to structure a web page.

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the web. HTML is the most important of the 3 languages every web developer must learn, because it is impossible to create a website only with CSS or JavaScript.

The technology consists of a series of elements (enclosed by <tags>) that tell the browser how to display the content.

For example, <h1> to <h6> would make headings; a <p> makes a paragraph; a <br> adds a line break between content; The <img> element adds images when using the attribute src and a <div> creates sections in the document. These were some of the elements used in this site.

With HTML, the order of the content is displayed according to the order it has been written in.

HTML can embed programs written in a scripting language such as JavaScript, which affects the behavior and content of web pages. Inclusion of CSS defines the look and layout of content.


How Does HTML Work?


Below, you can find an example of how a HTML code is written and how it is rendered by the browser (without CSS).

The example shows a number of tags being used and how the page is rendered on the right.

  • The !DOCTYPE html declaration tells the browser to render the document as HTML5.
  • All the content of the page should be inside the <html>.
  • The <head> is where information about the page, like encoding, language, description etc, can be added using the <meta> as well as linking and external CSS file with <link> and the <title> of the document (which only appears on the browser tab). Apart from the title, nothing else is displayed in the rendered page.
  • Within the <body>, tags like <h1> (heading 1) and <p> (paragraph) are used to create the content.
  • Some tags, such as <br> (line break) or <hr> (horizontal rule – it adds a line separator) do not have a closing tag (</>).
  • The <a> with the attribute href creates a hyperlink that directs the user to another page or website.
