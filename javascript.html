CLIENT-SIDE PROGRAMMING

JAVASCRIPT

JavaScript (JS for short) is the world's most popular programming language, well-known as the scripting language for Web pages. It is also used for various non-browser environments like Node.js.

Although JavaScript has "Java" in its name, the two programming languages are completely different from each other, having very different syntax, semantic, and use. JavaScript was invented by Brendan Eich in 1995, and became an ECMA standard in 1997.

This technology is used to add interactivity, dynamic styling and animation to websites. Alongside HTML5 and WebGL (Web Graphics Library), JavaScript can also be used to make games you can play on your browser.

JavaScript is the dominant client-side scripting language of the Web, with 97% of websites using it for this purpose. Scripts are embedded in or included from HTML documents and interact with the DOM (Document Object Model).

All major web browsers have a built-in JavaScript engine that executes the code on the user's device.

The use of JavaScript has expanded beyond its web browser roots. JavaScript engines are now embedded in a variety of other software systems, both for server-side website deployments and non-browser applications.


Examples of JavaScript


JavaScript can be used for a number of different interactions within the browser. Below, you can find some examples:

  • Loading new content without reloading the entire page, such as social media websites;
  • Animation of page elements, such as resizing, fading in/out;
  • Interactive content, like games or buttons;
  • Validating input collected from a form before they are submitted;
  • Collection of information about the user's online behaviour for ads personalisation.

The Light Bulb





But what is happening?


With only three lines of code, we can create this simple JavaScript example in which it simply changes the picture of the light bulb off to one that is on and vice-versa. For example:

On the JavaScript File:


function lightOff() {

document.getElementById("light-bulb").src='images/light-off.gif';

}

function lightOn() {

document.getElementById("light-bulb").src='images/light-on.gif';

}


On the HTML file:


<button onclick="lightOff()">TURN ON THE LIGHT</button>

<img id="light-bulb" src="images/light-on.gif" style="width:100px;">

<button onclick="lightOn()">TURN OFF THE LIGHT</button>


For this example, I created two functions on the external .js file – lightOff() and lightOn(). Then, I used the line getElementById to select the picture I placed between the two buttons. I used a picture of the light off for the first function and light on for the second. After that, I called the function with the onclick Event inside the buttons. By doing that, every time the user clicks the buttons, the picture of the light bulb will change, creating the efect of turning it on and off.
